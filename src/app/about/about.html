À propos de Diskover

Diskover est une application Angular de découverte musicale. Elle permet d'explorer la discographie des artistes.

L'API utilisée est Deezer. Elle expose les endpoints suivants :

  • Recherche d'artiste par nom : /search/artist?q={{ '{' }}nom{{ '}' }}
  • Top tracks d'un artiste (avec albums) : /artist/{{ '{' }}id{{ '}' }}/top?limit=50
  • Morceaux d'un album : /album/{{ '{' }}id{{ '}' }}
